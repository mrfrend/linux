===Настройка обмена данными с другими системами===
=Общая папка между ВМ и ПК=
Нужен сетевой мост!!
1. Добавляем новый адаптер "Сетевой мост" в настройках ВМ
2. Создаем новую папку на ПК и добавляем ее к ВМ в пункте "Общие папки"
3. Добавляем пустой "оптический привод" если отсутствует
3. Устанавливаем virtualbox-guest-utils:
sudo pacman -S virtualbox-guest-utils
4. Добавляем юзера в группу vboxsf:
sudo usermod -aG vboxsf $USER
5. Включаем vboxservice:
systemctl enable vboxservice
6. Перезагружаем ВМ через терминал:
sudo reboot


---
1. Проверьте список общих папок:
bashVBoxControl sharedfolder list
2. Проверьте, есть ли примонтированные папки:
bash# Проверьте стандартное расположение
ls -la /media/

# Проверьте монтирование
mount | grep vboxsf

# Проверьте наличие папок sf_*
ls -la /media/sf_*



4. Если папка настроена, но не видна в Linux:
bash# Создайте директорию для монтирования
sudo mkdir -p /mnt/shared

# Попробуйте примонтировать вручную
# Замените "ИмяПапки" на имя из настроек VirtualBox
sudo mount -t vboxsf ИмяПапки /mnt/shared

# Если работает, проверьте доступ
ls -la /mnt/shared

=Общая папка между ВМ и ВМ=
Нужен сетевой мост!!
1. Создаем общую папку между ПК и ВМ.
1. Выключаем ВМ.
2. ПКМ на ВМ и нажимаем "Клонировать" -> выбираем "сгенерировать новые MAC-адреса..."-> нажимаем "Далее" -> нажимаем "Готово".
3. Запускаем обе ВМ.
4. Открываем проводник и в левой панели внизу у обоих ВМ должна находиться общая папка.

===Отключение композиции рабочего стола===
Открываем Settings -> переходим в пункт Accessibility -> переходим в пункт Seeing -> нажимаем на тумблер Animation Effects (отключаем).

===Отключение масштабирования изображения при высоком разрешении экрана===
Открываем Settings -> переходим в пункт Displays -> выбираем большое Resolution (например 2048x1152) -> появляется параметр Scale, выбираем 100% -> нажимаем Apply
